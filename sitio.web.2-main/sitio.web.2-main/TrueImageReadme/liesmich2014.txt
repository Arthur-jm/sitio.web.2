Falls True Image neu aktiviert werden möchte:
"Aktivieren" anklicken, dann "Von einem anderen Computer aus aktivieren"
und irgendeinen Aktivierungscode eingeben, z.b. 1234
(dank der gepatchten "libcrypto10.dll" wird alles akzeptiert)

Wer das durchprobieren will:
? anklicken -> Seriennummer wechseln -> denselben oder anderen key eingeben...
(dadurch werden auch die dateien für die Notfallmedien aktualisiert,
falls jemand damit Probleme hat)
